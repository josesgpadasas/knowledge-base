Information and resources for the FishCRRM 1.1 Knowledge Base

About this Knowledge Base

The FishCRRM 1.1 Knowledge Base is a comprehensive digital platform designed to centralize and organize information, resources, and data related to the Fisheries and Coastal Resilient Resource Management (FishCRRM) Sub-component 1.1 of the Philippine Fisheries and Coastal Resiliency (FishCoRe) Project.

This knowledge base serves as a centralized repository for project documentation, implementation structures, contact directories, activity records, reference materials, and coverage area information. It provides stakeholders, project personnel, and partners with easy access to essential project information and resources.

How to Use This Knowledge Base

Navigation

Use the top navigation bar to access different sections of the knowledge base, including Implementation Structure, FMA Profile, FMA Municipalities, Landing Centers Map, Activities, Directory, and References.

Global Search

Use the global search bar in the navigation to search across all data and content in the knowledge base.

Filters

Each section includes filters to help you find specific information quickly. Use filters by year, category, component, FMA, region, province, or other relevant criteria. The FMA Municipalities page includes multi-level filtering with dependent province selection, while the Landing Centers map offers comprehensive filtering with real-time marker updates.

Export Data

Export directory information as CSV files for offline use or further analysis.

Features

  • Implementation Structure: View the organizational structure of the FishCRRM Component at National, FMA, and Regional levels.
  • FMA Municipalities: Comprehensive list of municipalities with advanced filtering by FMA, Region, and Province. Features include pagination, summary statistics with side-by-side tables showing municipalities by Region and FMA, and NSAP monitoring indicators.
  • Landing Centers Map: Interactive map visualization of landing centers with multiple filter options (FMA, Region, Province, Fishing Ground, and search by city/municipality or landing center name). Includes base map controls, marker clustering, and real-time filtering with count display.
  • Activities: Browse activities, trainings, and coordination meetings with timeline view and filtering options.
  • Directory: Access contact information for Internal, External, and NPMO personnel with search and filter capabilities.
  • References: Download reference documents and materials organized by category.
